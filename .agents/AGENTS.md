# Workspace Rules & Preferences

## 1. Design & Animations (Hallmark + React Bits)
- Enforce the `hallmark` skill to audit all new visual layout changes. Ensure pages have structural variety, HSL gradients, and skeuomorphic details.
- When generating animations or background effects, reference the `@react-bits` registry and use high-quality animated, interactive React components (e.g. customized backgrounds, magnetics, text effects).
