# Hallmark Anti-AI-Slop Design Skill
Based on usehallmark.com, this skill prevents code generation from falling into common AI-generated templates (centered hero, generic gradients, 3 feature columns, plain layouts).

## Core Design Principles

1. **Avoid Layout Slop**:
   - Do NOT use the standard "centered title + subtitle + CTA + 3-box feature list" layout pattern.
   - Use asymmetric grids (e.g., 5-column / 7-column splits, editorial layouts).
   - Integrate structured sidebars, deck cards, or step-sequencer grids rather than plain list items.

2. **Visual Variety & Typography**:
   - Use premium typography sizes and styles (e.g., extremely large mono-space status numbers, black font weights, variable sizes).
   - Use physical/skeuomorphic details (like VST knobs, mounting screws, audio patchbay ports, and glowing LED indicators).

3. **Color Guidelines**:
   - Avoid standard primary blue/red. Use tailored HSL gradients, dark modes, and soft, warm amber/green/indigo/purple hues.
   - Employ glow filters (`drop-shadow-[0_0_Xpx_rgba(...)]`) for active elements to give them an analog synth feeling.
